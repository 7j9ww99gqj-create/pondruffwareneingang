# Pondruff / WE Wareneingangs-Tool Demo

## Lokal starten

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Community Cloud

1. ZIP entpacken
2. Ordner in ein GitHub Repository hochladen
3. Bei Streamlit Cloud `app.py` als Startdatei auswählen

## Enthaltene Demo-Funktionen

- Dashboard im Pondruff/WE Design
- Neuer Wareneingang mit 3 Pflichtfotos: Lieferschein, Bauteile, Verpackung
- Zusatzdaten: Artikelnummer, Bezeichnung, Menge, Maße, Polieren, Beschichtung, Schichtdicke, Hinweise
- Beispiel-Wareneingänge mit Bildern
- Archiv und Detailansicht
- Bürobereich mit Tabelle und Copy/Paste-Datenblock für WISO
- KI-Suche als realistische Demo mit Trefferanzeige
- Präsentationsbilder direkt in der App

Hinweis: Diese Version nutzt keine echte Cloud, keine echte OCR und keine echte KI. Sie ist als klickbare Demo und Konzept-Prototyp gedacht.
